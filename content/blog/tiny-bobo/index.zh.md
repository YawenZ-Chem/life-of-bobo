感恩节假期，我飞去纽约接光头，再带他一起回波士顿。

和猫舍的繁育人交接时，对方刚打开汽车后备箱，光头就在混着冰雪的寒风里扯开嗓子疯狂大叫。

我简直震惊了：这么小的一只猫，怎么能发出这么大的声音？

从酒店到机场，再从机场回到波士顿的家，他就这样一路叫个不停。猫舍大概是把整窝小猫里最话唠的那只交给我了。

不过我转念一想，能叫得如此响亮，说明他的身体应该很健康。

好像也不算是一件坏事。

回到公寓以后，光头立刻绕着每个房间到处跑，一边跑一边继续叫。

后来我才意识到，他是在找猫砂盆。

我伸出手指，一路给他指到猫砂盆前。他马上跳了进去。从那一刻开始，他便永远记住了猫砂盆的位置。

看来他还挺聪明。

最开始，光头只有两个巴掌那么大。猫舍给他穿的小衣服，是用一只旧袜子改造的。回家以后，我也找来一只袜子，亲手给他剪了另一身衣服。

{{< /story-row >}}

其实，光头并不喜欢穿衣服。

不管衣服有多合身，好像都会限制他的活动。所以大部分时候，我也很愿意让他光着身子，在公寓里自由自在地到处乱跑。

让我觉得神奇的是，从到家的第一刻开始，他就寸步不离地跟着我。

他似乎完全不需要花时间接触我、研究我，再慢慢决定我是否值得信任。

只要我在书桌前坐下，他就会立刻跳到我的腿上。先端端正正地坐一会儿，再慢慢变成一只小小的母鸡蹲，最后沉沉睡去。

{{< /story-row >}}

那时的光头实在太小了。即使把身体坐得笔直，脑袋也只能勉强超过桌子的边缘。

每天我坐在电脑前工作，他就仰着脑袋，安安静静地看着我。

有时候，我不知道他只是想和我待在一起，还是也真的很想参与工作。

如果他能顺便替我写实验报告，那该多好啊。

{{< /story-pair >}}

最开始，我特别不放心让光头独自在客厅里睡觉。

理论上，作为一只独立的小秃猫，他应该拥有一个属于自己的睡觉空间。

但是他实在太小了，公寓里的温度也不够高。所以来到家的第一个晚上，我就把他带上了我的床。

{{< /story-row >}}


{{< story-row
  src="hold-my-paw.jpg"

从那以后，光头便理所当然地认为：我的床就是他应该睡觉的地方。

直到现在，我也不知道他究竟是聪明、固执，还是明知故犯地钻了一个规则漏洞。

每天晚上，只要我关掉灯、走进卧室，他都会立刻屁颠屁颠地跟过来。

{{< /story-row >}}


{{< story-row
  src="tiny-house-tiny-tenant.jpg"

其实，我给光头买过一个自认为既温暖又可爱的猫窝。

我还特意把它放在卧室里，想着我们可以继续当睡在同一个房间里的室友。

然而，光头还是坚持睡在我的床上。

最后，我只好把猫窝搬到书桌旁边。只要工作时还能盯着我，他倒是愿意蜷在里面睡上一会儿。

有时候，他睡醒以后还会忘记自己的舌头依然露在嘴巴外面。

{{< /story-row >}}

我躺在沙发上玩手机时，光头有时会用一种十分怪异的姿势歪倒在旁边。

但只要我开始全神贯注地看手机，他很快就会心生嫉妒，立刻跑过来，把我的手从手机旁边抢走，再用力把自己的秃脑袋往我的掌心里拱。

他求撸的决心总是让我震惊。

这么小的身体，究竟是怎么爆发出如此大的力气，把脑袋往我手里顶的？

{{< /story-pair >}}

很快，我就完全习惯了光头形影不离的陪伴。

有些时候，我必须一边抱着他，一边继续工作，所以只剩下一只手可以打字。

我抱光头的姿势，也从来不像是在抱一只猫。

他的两只前爪会搭在我的胸前，有时候甚至还会抱住我的脖子，看起来更像是我抱着一个小小的人类幼崽。

{{< /story-row >}}


{{< story-row

还有一些时候，我必须把双腿调整成一个非常特别的形状，让光头可以稳稳地睡在中间。

工作时，我偶尔会伸手戳一戳睡着的光头。

戳起来，就像在戳一颗温暖的桃子。

那时的我特别羡慕光头拥有的这种超能力：无论身处什么地方，他都能随时睡着；醒来以后，又能立刻恢复精神，重新变得活力十足。
